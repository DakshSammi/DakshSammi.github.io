## 🚀 Getting Started (3 Steps)

### Step 1: Personalize
Edit these files with YOUR information:
- `components/hero.tsx` - Your name, photo, title
- `components/about.tsx` - Your story and interests
- `components/experience.tsx` - Your jobs and projects
- `components/skills.tsx` - Your technical expertise
- `components/publications.tsx` - Your research/writing
- `components/contact.tsx` - Your social links

See `CUSTOMIZE.md` for detailed instructions.

### Step 2: Test Locally
```bash
npm run dev
# Open http://localhost:3000
```

### Step 3: Deploy
```bash
# Push to GitHub
git push origin main

# Deploy to Vercel (auto-deploys from GitHub)
# Visit vercel.com → Import project → Done! 🎉
```
